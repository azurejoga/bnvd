Sobre o BNVD

Democratizando o acesso à informação de segurança cibernética no Brasil

Nossa Missão

O BNVD foi criado para preencher uma lacuna crítica no ecossistema de segurança cibernética brasileiro: a falta de um banco de dados nacional de vulnerabilidades em português. Nossa missão é democratizar o acesso à informação sobre vulnerabilidades de segurança, fornecendo dados precisos, atualizados e compreensíveis para profissionais de TI, desenvolvedores e organizações brasileiras.

O que Oferecemos

  • Banco de dados completo de CVEs em português
  • Busca avançada por múltiplos critérios
  • Informações de severidade CVSS traduzidas
  • Classificação por tipos de fraqueza (CWE)
  • Atualizações automáticas da base NVD
  • Interface responsiva e intuitiva
  • Filtros por vendor, produto e ano
  • Detalhes técnicos completos

Tecnologia

O BNVD é desenvolvido utilizando tecnologias modernas e confiáveis:

Backend
  • Python 3.x
  • Flask Framework
  • Integração com API NVD 2.0
  • Rate limiting e cache
Frontend
  • HTML5 semântico
  • Bootstrap 5
  • JavaScript vanilla
  • Design responsivo

Sobre as Traduções Automáticas

⚠️ Aviso Importante

Todas as descrições de vulnerabilidades são traduzidas automaticamente por máquina, não por humanos.

Como Funciona Atualmente
  • Tradução em tempo real a cada visualização
  • Processo lento que afeta a velocidade do site
  • Possíveis imprecisões em termos técnicos
  • Google Translate como fonte principal
Como Você Pode Ajudar
  1. Baixe os XMLs dos anos 2002-2025 da NVD
  2. Traduza para português brasileiro
  3. Envie as traduções para a NVD
  4. A NVD incorpora no site oficial
  5. Acessamos diretamente via API
Baixar XMLs da NVD
Por que Tradução Humana é Melhor?

Traduções humanas garantem precisão técnica, contexto adequado e terminologia correta. Com traduções oficiais da NVD, poderemos eliminar a tradução em tempo real e melhorar significativamente a velocidade do site.

Fonte dos Dados

Todos os dados de vulnerabilidades são obtidos diretamente da National Vulnerability Database (NVD) do NIST (National Institute of Standards and Technology) dos Estados Unidos, que é a fonte oficial e mais confiável de informações sobre vulnerabilidades CVE mundialmente.

Atualização: Nossa base de dados é sincronizada regularmente com a NVD para garantir que você tenha acesso às informações mais recentes sobre vulnerabilidades de segurança.

Como Usar

1. Busca Simples

Use a busca rápida na página inicial para encontrar vulnerabilidades por CVE ID, vendor ou ano.

2. Busca Avançada

Na página de busca, utilize múltiplos filtros para refinar seus resultados.

3. Análise Detalhada

Clique em qualquer CVE para ver informações detalhadas incluindo métricas CVSS e produtos afetados.

4. Compartilhamento

Compartilhe links diretos para vulnerabilidades específicas com sua equipe.
